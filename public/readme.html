SDVX Helper Realtime Battle

リアルタイム対戦システム ドキュメント

📋 システム概要

基本機能

このシステムは、コナステ版SOUND VOLTEXのゲーム画面をキャプチャするクライアントアプリと連携し、 複数人での対戦に使用することができるリアルタイム対戦システムです。

  • BPL風2人対戦ビュー: リアルタイムスコア比較とバー表示による優劣確認
  • 多人数ランキングビュー: 3人以上での利用を想定したランキング形式の表示画面
  • リアルタイム集計: スコア送信と同時にランキングが更新され、配信での利用も可能。

動作環境・性能について

性能制限について: 現在このシステムはRender.comの無料枠で動作しており、性能はそれほど高くありません。 同時アクセス10人程度での利用を想定して設計されています。

自分専用システムの構築について

サーバー側のリポジトリは公開予定のため、Render.comの無料枠を他に使っていない方であれば、 ご自身専用のシステムを構築することが可能です。

  • サーバ側レポジトリ
  • クライアント側でシステムのURLを変更できるため、独自環境での運用も可能

クライアントアプリについて

現在: 専用バイナリとして提供
将来的に: sdvx_helperの組み込み機能として統合される可能性があります

クライアントアプリは、ゲーム画面から自動的にスコア情報を読み取り、 このWebシステムにリアルタイムで送信する機能を持っています。

📱 クライアントアプリのインストール方法

注意: クライアントアプリの詳細な情報については、[適切なリンクまたは説明を追加してください]

システム要件

項目 要件
OS Windows10/11
メモリ 8GB以上を推奨
OBS Studio 28以降が必要

インストール手順

  1. ダウンロードおよびインストール
    配布先からsdvx_arena.zipをダウンロードし、好きなフォルダに解凍する。
    OneDriveに管理されているフォルダ(Desktop, MyDocumentなど)は避けたほうがよい。
  2. OBSおよびOBSWebsocketの準備
    OBSをインストールしておく。OBSWebsocketも使えるようにしておく。 クライアントアプリにwebsocketのポートとパスワードを入力する必要があるので控えておく。
  3. クライアントの初回設定
    起動後に設定画面を開く。

    OBSWebsocketのポート及びパスワードを入力する。
    接続ボタンをクリックし、画面取得に使うソース名を選択しておく。
    必要に応じて、画面の向きやファイル保存の有無について選択する。

    保存をクリックして反映させる。

📖 サイトの使い方

基本的な流れ

  1. サイトに接続
    ユーザ名を入力して接続をクリックする。
  2. 部屋の作成
    新たに部屋を作成する場合、部屋名・ルール(通常スコア/EXスコア)を入力して部屋の作成をクリックする。必要に応じて入室時のパスワードを設定する。

    作成済みの部屋をクリックして入室する。
    現在、クライアント側に部屋の作成機能を搭載していないため、PCやスマホで作成しておく必要がある。
  3. 対戦画面の確認
    部屋内のメニューから2人対戦ビューや多人数ビューのボタンをクリックし、各ビューを確認する。
    それぞれ別々のURLを振っているため、OBSから表示することもできる。
  4. クライアントアプリからの接続
    注意: クライアントアプリのインストール及びOBS連携設定を済ませておいてください。
    アプリを起動し、必要に応じてプレーヤー名を入力してシステムに接続する。

    作成済みの部屋を選択して入室する。 あとはチャットなどを使って曲を決めておき、プレーを行う。
    全プレイヤーからのスコア送信が完了した時点で1曲分のポイントが集計されます。
